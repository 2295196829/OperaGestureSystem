

本作品是一款基于Leap Motion，结合了虚拟现实技术和人机交互技术的创新互动应用，旨在通过沉浸式的体验让玩家深入了解和体验中国戏曲文化。

## 开发环境&硬件需求
- **Unity Editor 2022.3.15f1c1**
- Visual Studio 2022
- Leap Motion Controller

![Leap Motion Controller](https://www.ultraleap.com/assets/images/devices/lmc-device-1.png)

## 安装
- Gesture Detect:[Ultraleap Track Service](https://www.ultraleap.com/downloads/leap-controller/)

- Develop Docs：[LeapMotion in Unity](<https://docs.ultraleap.com/xr-and-tabletop/xr/unity/getting-started/index.html>)
